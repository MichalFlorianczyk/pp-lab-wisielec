Gra w Wisielca

Opis
Gra w Wisielca to klasyczna gra słowna, w której gracze próbują odgadnąć ukryte słowo poprzez zgadywanie liter. Gracz ma ograniczoną liczbę prób, a każda błędna litera przybliża go do przegranej.

Funkcje
- Wybór losowego słowa z wbudowanej bazy słów.
- Możliwość dodawania, usuwania i edytowania słów w bazie.
- Wybór poziomu trudności:
  - Łatwy (6 prób)
  - Średni (12 prób)
  - Trudny (24 próby)
- Wyświetlanie statystyk:
  - Liczba rozegranych gier
  - Liczba wygranych gier
  - Liczba przegranych gier
  - Średnia liczba prób potrzebna do wygranej gry

 Instalacja
1. Sklonuj repozytorium na swój lokalny komputer:
   bash
   git clone https://github.com/MichalFlorianczyk/pp-lab-wisielec
   
2. Przejdź do katalogu projektu:
   
   cd hangman-game
   
3. Upewnij się, że masz zainstalowaną JDK w wersji 8 lub nowszej.

Uruchomienie
1. Otwórz projekt w preferowanym IDE (np. IntelliJ IDEA, Eclipse).
2. Skonfiguruj projekt jako projekt Maven, jeśli używasz Mavena.

Użycie
1. Po uruchomieniu gry, zobaczysz menu główne:
      Witaj w grze w wisielca!
   1. Zagraj
   2. Dodaj/Usuń/Edytuj słowo
   3. Statystyki
   4. Wyjście

2. Wybierz odpowiednią opcję:
   - 1: Rozpocznij nową grę. Zostaniesz poproszony o wybór poziomu trudności, a następnie będziesz zgadywać litery.
   - 2: Zarządzaj bazą słów (dodawaj, usuwaj, edytuj i wyświetlaj listę słów).
   - 3: Wyświetl statystyki gier.
   - 4: Zakończ grę.

Zarządzanie Bazą Słów
1. Wybierz opcję 2 z menu głównego.
2. Zobaczysz podmenu zarządzania słowami:
   
   1. Dodaj słowo
   2. Usuń słowo
   3. Edytuj słowo
   4. Wyświetl listę słów
   5. Powrót

3. Wybierz odpowiednią opcję:
   - 1: Dodaj nowe słowo do bazy.
   - 2: Usuń istniejące słowo z bazy.
   - 3: Edytuj istniejące słowo.
   - 4: Wyświetl listę wszystkich słów w bazie.
   - 5: Powrót do menu głównego.

4. Uwagi ogólne - program nie wychwytuje polskich znaków.
